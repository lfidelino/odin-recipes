# odin-recipes

[Project: Recipes](https://www.theodinproject.com/lessons/foundations-recipes) from [The Odin Project](https://www.theodinproject.com).

I have learned about how to setup a development environment, setup and use git, and create simple web pages with HTML.

I used simple HTML tags to complete this project. I also tried to commit changes as frequent and as timely as I can.

Further along the course, I was asked to go back to this project and [style the pages with CSS](https://www.theodinproject.com/lessons/foundations-css-foundations#practice).

View on [Netlify](https://lfidelino-odin-recipes.netlify.app/) or [GitHub Pages](https://lfidelino.github.io/odin-recipes/).
